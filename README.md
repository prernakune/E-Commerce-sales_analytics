# Project Overview

As part of my data analytics learning journey, I worked on an E-Commerce Sales Analysis project to understand sales performance, customer purchasing behavior, and business profitability. The objective was to transform raw sales data into meaningful insights that could help businesses make data-driven decisions.

# This project was completed using Python for data analysis, MySQL for business querying, and Power BI for dashboard development.

Tools Used
Python (Pandas, NumPy)
MySQL
Power BI
Dataset Overview

The dataset contains 5,000 e-commerce sales records with information related to:

Orders
Customers
Product Categories
Sales and Profit
Quantity Sold
Discounts
Regions and Cities
Payment Modes
Order Dates
Data Preparation

Before starting the analysis, I performed several data cleaning and preprocessing steps:

Examined dataset structure and data types
Checked for missing values and duplicate records
Converted Order Date into datetime format
Created Year, Month, Month Name, and Quarter columns
Calculated Profit Margin Percentage for profitability analysis
Key Metrics Analyzed
Total Sales
Total Profit
Total Orders
Average Order Value
Profit Margin %
Python Analysis

# Using Python, I explored the dataset and analyzed different business dimensions:

Sales by Category

Identified product categories contributing the highest revenue.

Sales by Region

Compared revenue performance across different regions.

Payment Mode Analysis

Analyzed customer preferences for different payment methods.

Profitability Analysis

Calculated profit margins to evaluate business performance.

Key Insights
North region generated the highest sales revenue.
Home Decor and Furniture were among the top-performing categories.
Net Banking contributed the highest share of sales among payment methods.
Sales performance varied across regions and categories.
Profitability differed across product categories, highlighting areas for business optimization.
